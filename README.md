# TicTacToe
Projet réalisé en Java avec Swing
L'application permet de jouer au TicTacToe contre l'ordinateur : le joueur choisit son pion avant le début de la partie puis l'ordinateur joue automatiquement en plaçant son pion aléatoirement sur une case de disponible

## Comment jouer
1. Lancer la classe Main
2. Depuis l'écran d'accueil, cliquer sur Jouer une partie
3. Choisir son pion : X ou O
4. Choisir la taille de la grille (3x3 à 9x9)
5. Cliquer sur Jouer pour démarrer la partie
6. Cliquer sur une case vide pour placer son pion
7. L'ordinateur joue automatiquement après chaque coup
8. En fin de partie, cliquer sur Nouvelle partie pour rejouer et recommencer depuis l'étape 3

## Fonctionnalités
- Jouer contre l'ordinateur
- Choisir son pion avant la partie : X ou O
- L'ordinateur joue aléatoirement sur une case libre
- Possibilité de lancer une nouvelle partie sans redémarrer l'application
- Taille de grille configurable de 3x3 à 9x9

## Règles du jeu
- Le pion X commence toujours
- Si le joueur choisit O, l'ordinateur commence la partie avec X
- Le but est d'aligner ses pions sur une ligne, une colonne ou une diagonale
- Pour une grille de taille N x N, il faut aligner N pions identiques pour gagner
